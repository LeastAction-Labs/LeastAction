  <ai_provider_pattern>
  This action type generates code that calls AI provider APIs using LangChain. The pattern is:

  1. Extract connection credentials (api_key, model, token_limit) from least_action_action_object.get('connection')
  2. Initialize the LangChain chat model (e.g., ChatAnthropic, ChatOpenAI)
  3. Build message chain using SystemMessage, HumanMessage, AIMessage
  4. Invoke the model and process the response
  5. Return True on success, False on failure

  Supported LangChain providers and imports:
  - Anthropic: from langchain_anthropic import ChatAnthropic
  - OpenAI: from langchain_openai import ChatOpenAI
  - Google: from langchain_google_genai import ChatGoogleGenerativeAI

  Common message imports:
  - from langchain_core.messages import SystemMessage, HumanMessage, AIMessage

  The connection object will always contain:
  - api_key: The API key for the provider
  - model: The model name (e.g., "claude-sonnet-4-20250514", "gpt-4o", "gemini-pro")
  - token_limit: Maximum tokens for the response

  Optional connection fields:
  - temperature: Model temperature (default 0.0)
  - timeout: Request timeout in seconds (default 60)
  </ai_provider_pattern>
